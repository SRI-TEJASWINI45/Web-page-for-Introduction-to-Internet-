INTRODUCTION TO INTERNET


Internet:

Internet is a network of networks which consists of inter-connected networks.It is a worldwide network of computers that uses common communication standards and interfaces to provide the physical backbone for number of interesting applications.


World Wide Web:

WWW is a abbreviation of ‘ World Wide Web’.The <>BWWW or simple web is a way of accessing info over the Internet. It contains millions of documents or ‘bunch of pages’ called web pages. The web uses the HTTP (Hyper Text Transfer Protocol) for the transfer of these documents over the Internet.Sir Tim Berners- lee, along with Robert Cailliau, invented the WWW while working at CERN. (European Council For Nuclear Research) in 1990.


Features of World Wide Web:

AVAILABILITY OF CONTENT:

The info on the web is available 24X7. The is updated in real time and made accessible to the users of the web. Most of the websites are open to everyone.

UNIVERSAL STANDARD:

The web uses a common format for writing wed content-HTML and a common HTTP for accessing the web info. HTTP is the standard protocol that allows the transfer of hypertext documents on the World Wide Web.

HYPERTEXTS AND HYPERLINKS:

Hypertexts are pointers to info or HTML pages. Hyperlinks allow the web users to choose their paths across info on the web.

QUICK SEARCH:

The Search Engines on the web allow an easy and efficient search process for info. This data facilities the quest for info by millions of users across the globe.

FOCUS ON SHARED INTERESTS:

The WWW brings the people with common interests together, from different locations of the world. Users can exchange resources as well as ideas, and can also benefit from them.
